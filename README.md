## Currently shipping

- **[QuantaLang](https://github.com/HarperZ9/quantalang)** — effects-oriented systems language. Rust compiler, multi-backend codegen (C · HLSL · GLSL · SPIR-V · LLVM · WASM · x86-64 · ARM64). MIT.
- **[Quanta Universe](https://github.com/HarperZ9/quanta-universe)** — 43-module ecosystem written in QuantaLang: rendering engines, color science, algorithmic trading, kernel work, developer tools. Alpha.
- **[Quanta Color](https://github.com/HarperZ9/quanta-color)** — Python color science library. 15 color spaces, 12 tone mappers, CIECAM02/CAM16, spectral rendering, PyQt6 GUI.
- **[Calibrate Pro](https://github.com/HarperZ9/calibrate-pro)** — Sensorless display calibration for Windows. 58-panel hardware database, DDC/CI, 3D LUT, ICC v4 profiles, optional i1Display3 measurement.
